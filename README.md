# Django CRUD Blog Starter

A beginner-friendly Django CRUD project for managing blog content with template-based pages.

## Highlights
- CRUD workflow for blog-style entries
- Simple template structure for rapid iteration
- Clean starter layout for learning Django fundamentals

## Tech Stack
- Python
- Django
- HTML

## Run Locally
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
